Detailed Description

An exception that is thrown when a file is not found where it should be. This can mean that the key being used to search for a file is incorrect, or that the file is not registered. If this exception is thrown, it is recommended that you ensure that you have the correct key(s) being used to search for the file and that the file exists within the sandbox.
