Detailed Description

A bunch of utilities that can interact with the OS in a mostly cross-platform way. Alot of this simply wraps around the Windows API and the Linux API. The utilities try to remain as close to how they originaly were written as possible, but some changes have been made to make them more cross-platform.

Warning
This file shall NEVER include raylib because raylib will have name conflicts with the Windows API.
