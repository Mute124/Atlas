\doxysubsection{Detailed Description}
This is the interface for the user\textquotesingle{}s project. This is the only interface that \doxylink{namespace_atlas}{Atlas} will call. 

This is an {\bfseries{abstract interface singleton}} that all projects must implement and set the {\itshape \doxylink{class_atlas_1_1_i_project_1_1_project_reference}{Project\+Reference}} singleton to their project. In terms of functionality, this interface just holds the functions and variables that \doxylink{namespace_atlas}{Atlas} will call (besides some initialization). \begin{DoxyNote}{Note}
If this boilerplate code is not done, \doxylink{namespace_atlas}{Atlas} will not know what to run. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
{\bfseries{Do not directly call window code in here or your project will crash!}} 
\end{DoxyWarning}
