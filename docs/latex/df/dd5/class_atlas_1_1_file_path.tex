\doxysubsection{Detailed Description}
This class represents a file path. Since the file system is sandboxed, this contains the {\bfseries{full path}} and the {\bfseries{sandboxed path}}. A sandboxed path is a relative to the "{}game"{} dir. For example, if a file was in the "{}game/data"{} folder, the path would be "{}data"{}. This is to make usage of the file system much easier. When constructed, the sandboxed path is automatically resolved. 

A full path is the exact location on the disk. This should NOT be exposed to the modding API as it is extremely dangerous and makes it easy for malicious mods to attack.
