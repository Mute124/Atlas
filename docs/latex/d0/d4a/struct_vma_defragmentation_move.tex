Temporary allocation pointing to destination memory that will replace {\ttfamily src\+Allocation}. 

\begin{DoxyWarning}{Warning}
Do not store this allocation in your data structures! It exists only temporarily, for the duration of the defragmentation pass, to be used for binding new buffer/image to the destination memory using e.\+g. \doxylink{group__group__alloc_ga7748e60ccc54e240777a848cb260bb19}{vma\+Bind\+Buffer\+Memory()}. \doxylink{group__group__alloc_gabbc69db60f82972ddffaceb6653e344a}{vma\+End\+Defragmentation\+Pass()} will destroy it and make {\ttfamily src\+Allocation} point to this memory. 
\end{DoxyWarning}
