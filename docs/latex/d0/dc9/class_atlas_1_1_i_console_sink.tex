\doxysubsection{Detailed Description}
This is NOT what will print to the terminal, rather this is an interface where the logger will pass logging messages to. Use this for in-\/game consoles or something like that. 

This is a sink for message logging. \begin{DoxyRemark}{Remarks}
This is used when the \doxylink{class_atlas_1_1_logger}{Logger} class recieves logs. Afterwards, it is forwarded to the \doxylink{class_atlas_1_1_i_console_sink}{IConsole\+Sink}. 
\end{DoxyRemark}
\begin{DoxyNote}{Note}
This can be used for in-\/game consoles or something like that.
