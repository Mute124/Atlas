\doxysubsection{Detailed Description}
Scheduler for running tasks on separate threads. It will take given tasks, and distribute them across threads. 

\begin{DoxyPrecond}{Precondition}
\doxylink{namespace_atlas}{Atlas} must have multi-\/threading enabled for this to even be used.
\end{DoxyPrecond}
\begin{DoxyNote}{Note}
Everything is thread-\/safe.

This class {\bfseries{CANNOT}} be copied.
\end{DoxyNote}
@important This class does not ensure the proper order of execution of tasks, so be careful when using it.
