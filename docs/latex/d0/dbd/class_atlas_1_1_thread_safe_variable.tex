\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T\+\_\+\+VARIABLE\+\_\+\+TYPE, typename T\+\_\+\+MUTEX = std\+::mutex, typename T\+\_\+\+LOCK\+\_\+\+GUARD = std\+::lock\+\_\+guard$<$\+T\+\_\+\+MUTEX$>$$>$\newline
class Atlas\+::\+Thread\+Safe\+Variable$<$ T\+\_\+\+VARIABLE\+\_\+\+TYPE, T\+\_\+\+MUTEX, T\+\_\+\+LOCK\+\_\+\+GUARD $>$}
This class wraps a variable (specified by T\+\_\+\+VARIABLE\+\_\+\+TYPE) and makes it thread-\/safe. 

When \doxylink{class_atlas_1_1_thread_safe_variable_a4007f83735117deb04971ae309f5f68f}{Thread\+Safe\+Variable\+::execute()} is called, it will lock this variable and execute the function that is passed to it. Furthermore, it will also lock when the \doxylink{class_atlas_1_1_thread_safe_variable_a7840da777c194e6f4b0bc78a163179e0}{set()} function, is called.

\begin{DoxyWarning}{Warning}
The variable that is wrapped by this class must have a default constructor. If the variable does not have a default constructor, then this class will throw an exception (Missing\+Default\+Constructor\+Exception).
\end{DoxyWarning}
