\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T\+\_\+\+VARIABLE\+\_\+\+TYPE, typename T\+\_\+\+MUTEX = std\+::shared\+\_\+mutex$>$\newline
class Atlas\+::\+Thread\+Safe\+Variable$<$ T\+\_\+\+VARIABLE\+\_\+\+TYPE, T\+\_\+\+MUTEX $>$}
This class wraps a variable (specified by T\+\_\+\+VARIABLE\+\_\+\+TYPE) and makes it thread-\/safe. 

When \doxylink{class_atlas_1_1_thread_safe_variable_a6a54dfc285406c6872bb9f3d0abe24c0}{Thread\+Safe\+Variable\+::execute()} is called, it will lock this variable and execute the function that is passed to it. Furthermore, it will also lock when the \doxylink{class_atlas_1_1_thread_safe_variable_a390f885a2796b7571d354b9c660adeca}{set()} function, is called.

\begin{DoxyWarning}{Warning}
The variable that is wrapped by this class must have a default constructor. If the variable does not have a default constructor, then this class will throw an exception (Missing\+Default\+Constructor\+Exception).
\end{DoxyWarning}
