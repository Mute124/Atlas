\doxysubsection{Detailed Description}
More detailed statistics than \doxylink{struct_vma_statistics}{Vma\+Statistics}. 

These are slower to calculate. Use for debugging purposes. See functions\+: \doxylink{group__group__stats_gad176279c8da8f50399674b2ef03ca866}{vma\+Calculate\+Statistics()}, \doxylink{group__group__stats_ga54b57aa8f511406711f4b918028d8f60}{vma\+Calculate\+Pool\+Statistics()}.
