\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{class_atlas_1_1_console_sink}{Console\+Sink}}$>$\newline
class Atlas\+::abstract$<$ Console\+Sink $>$}
This is NOT what will print to the terminal, rather this is an interface where the logger will pass logging messages to. Use this for in-\/game consoles or something like that. 

A class that acts as, well, a gate! It is essentially as a piece of machinery on an assembly line. Please do note, that this is a template class and is not meant to be used directly. Please see \doxylink{class_atlas_1_1_game_object_registry}{Game\+Object\+Registry} for more information and an example.

A drawable is anything that can be drawn to the screen. However, this does not include shaders as that is an entirely seperate system.

Same thing as a gate, but for game objects.

Represents an object that can be rendered in the screen. This is an abstract class that contains all the common properties of a game object.

You may implement this yourself to create your own game objects and drawing code, or you can inherit from \doxylink{class_atlas_1_1_game_object}{Atlas\+::\+Game\+Object} to use the default implementation. However, if you inherit from this class, be sure you have a basic understanding of Raylib\textquotesingle{}s and Open\+GL\textquotesingle{}s rendering system. If not, see https\+://learnopengl.\+com/ and https\+://www.\+raylib.\+com/ for a good place to start. In addition, it is not advised for you to inherit this yourself because it will require you to implement rendering code. However, it is recommended if your game requires a rendering system that is not provided by default. 

The IGame\+Object class is abstract and cannot be instantiated directly. Instead, you should inherit from this class and implement the pure virtual functions. As of version 0.\+0.\+3, the pure virtual functions are\+: \doxylink{class_atlas_1_1abstract_a69d0689a518724883cb9096773cef353}{render()}, \doxylink{class_atlas_1_1abstract_a2f47d74834f2351163c337d9d2801a0e}{texture()}, \doxylink{class_atlas_1_1abstract_a426a47734501228f84250147b4ee5154}{update()}, pre\+Physics\+Update(), \doxylink{class_atlas_1_1abstract_addec1f05420c246e42715ab0c3ca227e}{post\+Update()}, \doxylink{class_atlas_1_1abstract_af009ebb06d6e6d14419bdd8f7d78adac}{destroy()}, and \doxylink{class_atlas_1_1abstract_a10231a4cca55bad9d64b7e6b0e718d17}{cleanup()}.
