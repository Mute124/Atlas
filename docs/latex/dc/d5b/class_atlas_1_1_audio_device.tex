\doxysubsection{Detailed Description}
An audio device is a physical {\bfseries{OUTPUT}} device that can be used to play audio. If you are looking for a {\bfseries{INPUT}} device, see Audio\+Capture\+Device. If you are looking for documentation on the forms of {\bfseries{OUTPUT}} devices, see this class\textquotesingle{}s details documentation. 

\begin{DoxyNote}{Note}
Please also note that this class\textquotesingle{}s device type defaults to EAudio\+Device\+Type\+::\+Playback. 
\end{DoxyNote}




{\bfseries{Details}}\begin{adjustwidth}{1em}{0em}
Audio devices come in a few forms\+: 
\begin{DoxyItemize}
\item Playback\+:   An audio device that can be used to play audio.   
\item Duplex\+:   An audio device that can be used to both play and record audio.   
\item Loopback\+:   An audio device that can be used to record audio from another device.   
\end{DoxyItemize}\end{adjustwidth}
