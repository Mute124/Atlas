\doxysubsection{Detailed Description}
An audio device is a physical {\bfseries{OUTPUT}} device that can be used to play audio. If you are looking for a {\bfseries{INPUT}} device, see Audio\+Capture\+Device. If you are looking for documentation on the forms of {\bfseries{OUTPUT}} devices, see this class\textquotesingle{}s details documentation. 

\begin{DoxyRemark}{Remarks}
An audio device comes in a few forms\+: 
\begin{DoxyItemize}
\item EAudio\+Device\+Type\+::\+PLAYBACK\+\_\+\+DEVICE\+\_\+\+TYPE\+:   is the default and is an audio device that can be used to play audio.   
\item EAudio\+Device\+Type\+::\+DUPLEX\+\_\+\+DEVICE\+\_\+\+TYPE\+:   is an audio device that can be used to both play and record audio.   
\item EAudio\+Device\+Type\+::\+LOOPBACK\+\_\+\+DEVICE\+\_\+\+TYPE\+:   is an audio device that can be used to record audio from another device.   
\end{DoxyItemize}
\end{DoxyRemark}
