\doxysubsection{Detailed Description}
Represents an object that can be rendered in the screen and contains all the common properties of a game object. 

\begin{DoxyRemark}{Remarks}

\end{DoxyRemark}
\hypertarget{class_atlas_1_1_i_game_object_gameobjects-interface-uses}{}\doxyparagraph{\texorpdfstring{Class Use Cases}{Class Use Cases}}\label{class_atlas_1_1_i_game_object_gameobjects-interface-uses}
You may implement this yourself to create your own game objects and drawing code, or you can inherit from \doxylink{class_game_object}{Game\+Object} to use the default implementation. However, if you inherit from this class, be sure you have a basic understanding of Raylib\textquotesingle{}s and Open\+GL\textquotesingle{}s rendering system. If not, see \doxylink{}{https\+://learnopengl.\+com} or \doxylink{}{https\+://www.\+raylib.\+com} for a good place to start. In addition, it is not advised for you to inherit this yourself because it will require you to implement rendering code. However, it {\itshape is} recommended if your game requires a rendering system that is not provided by default. If you do implement your own rendering system, be sure to ensure that the \doxylink{class_atlas_1_1_renderer}{Renderer} used in the \doxylink{class_atlas_1_1_i_atlas}{IAtlas} derived class or \doxylink{class_atlas_1_1_atlas_engine}{Atlas\+Engine} class can be used with your code. 

\begin{DoxyNote}{Note}
The \doxylink{class_atlas_1_1_i_game_object}{IGame\+Object} class is abstract and cannot be instantiated directly. Instead, you should inherit from this class and implement the pure virtual functions. As of version 0.\+0.\+3, the pure virtual functions are\+: \doxylink{class_atlas_1_1_i_game_object_ac039e60ec710b378b1203b182713cbed}{render()}, \doxylink{class_atlas_1_1_i_game_object_a75b8e711b2bc32f44747e9c495fb9aa1}{texture()}, \doxylink{class_atlas_1_1_i_game_object_af51a5366c9d250e6ebe75ab7d1d8091a}{update()}, pre\+Physics\+Update(), \doxylink{class_atlas_1_1_i_game_object_a76d0ad0c7bdece628b9d4fd5cc4de731}{post\+Update()}, \doxylink{class_atlas_1_1_i_game_object_a0e4794c35b80eb92dbd3b0ac4626b4a5}{destroy()}, and \doxylink{class_atlas_1_1_i_game_object_aadb8ac4a4ee34905fb156c41929f24f6}{cleanup()}. 

This is an abstract class 
\end{DoxyNote}
