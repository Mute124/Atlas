\doxysubsection{Detailed Description}
A wrapper class for the usage of Vulkan as a rendering backend. This class handles the lifecycle of Vulkan and simplifies the usage of Vulkan to a simple API. 

\begin{DoxyPrecond}{Precondition}
The window must be set up and open prior to initializing Vulkan (and therefore this class!).
\end{DoxyPrecond}
\begin{DoxyWarning}{Warning}
This class requires the user\textquotesingle{}s device to support Vulkan!
\end{DoxyWarning}
