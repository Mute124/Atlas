class \mbox{\hyperlink{class_atlas_1_1_a_renderer}{Atlas\+::\+ARenderer}}
\begin{DoxyCompactList}\small\item\em A base class for all renderers. This class is responsible for {\bfseries{frontend}} rendering of your game. The rendering backend is responsible for the actual rendering. When using this class, you should inherit from it and implement the {\ttfamily init} and {\ttfamily update} methods. If you want this class to be multi-\/threaded by \doxylink{namespace_atlas}{Atlas}, you should make sure that the {\ttfamily init} and {\ttfamily update} methods are thread-\/safe. Additionally, make sure you set \doxylink{class_atlas_1_1_a_renderer_a6a77936bc8281507f5e205e8831a9852}{can\+Be\+Multi\+Threaded} to {\itshape true} if you want this to take effect. If you do not, \doxylink{namespace_atlas}{Atlas} will only run the {\ttfamily init} and {\ttfamily update} methods on the rendering thread instead of accounting for your renderer\textquotesingle{}s need to be multi-\/threaded. \end{DoxyCompactList}\end{DoxyCompactItemize}
