\doxysubsection{Detailed Description}
This class is used to make an object serializable and deserializable. This class is intended to be inherited by other classes. If you want to make a class serializable, inherit this class and you must override the \doxylink{class_atlas_1_1_serializable_a567102d8313786f3d2af41333fb39d6c}{serialize\+Variables} function. Do {\bfseries{NOT}} touch the \doxylink{class_atlas_1_1_serializable_ab89808083e4325daf8af8723aa203de3}{serialize} function because that is what boost uses to serialize the object. In the \doxylink{class_atlas_1_1_serializable_a567102d8313786f3d2af41333fb39d6c}{serialize\+Variables} function, you should set the values.
