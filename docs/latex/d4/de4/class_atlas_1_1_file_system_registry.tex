\doxysubsection{Detailed Description}
The file system registry. 

\begin{DoxyRemark}{Remarks}
The file system registry is a relatively complex class because it\+: 
\begin{DoxyItemize}
\item Keeps track of files.  
\item Handles loading of files  
\item Handles unloading of files  
\item Handles registration of files  
\item And just about everything else that a registry does  
\end{DoxyItemize}As an example of how to add a new file extension, see the \doxysectlink{index_file-loading}{File Loading}{4} document.
\end{DoxyRemark}
