\doxysubsection{Detailed Description}
This class represents a physical device that can provide haptic feedback to the user. Due to the sheer amount of ways haptic feedback can be provided, this class acts as a wrapper for the SDL\+\_\+\+Haptic object. 

\begin{DoxyPrecond}{Precondition}
As is with the rest of \doxylink{namespace_atlas}{Atlas}\textquotesingle{} haptic API, SDL must be initialized before this class can be used and the flag SDL\+\_\+\+INIT\+\_\+\+HAPTIC must be set.
\end{DoxyPrecond}

Opens the haptic device using the {\ttfamily Happtic\+Device\+ID} that was provided in the constructor. If the haptic device is already open, this function will essentially restart the haptic device by closing it and then opening it again. Keep in mind that this will cause all haptic effects to be lost. Since this class is built on SDL\textquotesingle{}s haptic feedback API, this function will open the haptic device using SDL\textquotesingle{}s API. Please do note that if the value of {\ttfamily s\+Current\+Haptic\+Devices\+Count} is equal to the value of {\ttfamily s\+Max\+Haptic\+Devices}, then this function WILL NOT open the haptic device unless the value of {\ttfamily s\+Max\+Haptic\+Devices} is set to -\/1.
