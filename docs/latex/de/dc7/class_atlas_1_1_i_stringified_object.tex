\doxysubsection{Detailed Description}
An interface for classes that want to be printable. 

\begin{DoxyRemark}{Remarks}
This is useful for debugging and logging as it provides a way to get a string representation of the object.
\end{DoxyRemark}
\begin{DoxyNote}{Note}
It is not recommended that you use this interface on everything as it may cause performance issues. 
\end{DoxyNote}
