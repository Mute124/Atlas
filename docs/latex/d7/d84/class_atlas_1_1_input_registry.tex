\doxysubsection{Detailed Description}
The input registry is used to register Input\+Actions (\doxylink{class_atlas_1_1_input_action}{Input\+Action}) and their callbacks by using a Laissez-\/faire system. 

\begin{DoxyNote}{Note}
Remember that this is a singleton, so you may use this statically. 
\end{DoxyNote}
\begin{DoxyRefDesc}{Bug}
\item[\mbox{\hyperlink{bug__bug000001}{Bug}}]When using functions that require rendering, the engine will crash because the input registry runs on a different thread. This can be fixed through a sync system. \end{DoxyRefDesc}
\begin{DoxySince}{Since}
v0.\+0.\+6
