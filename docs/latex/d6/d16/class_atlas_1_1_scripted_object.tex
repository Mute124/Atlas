\doxysubsection{Detailed Description}
An abstract class that provides the virtual functions needed to expose symbols to any potential mods/scripts. This class is supposed to be statically-\/typed, meaning it is supposed to be overriden instead of passing functions to it. 

This class is supposed to be inherited by any object that is supposed to be scripted. Among the features that this class should be able to provide are\+:


\begin{DoxyEnumerate}
\item Exposing symbols for mods to use.
\end{DoxyEnumerate}
