\doxysubsection{Detailed Description}
Serves as a search key/input category for the input system. 

\begin{DoxyWarning}{Warning}
Please pay attention when using this because any slight change in \doxylink{class_atlas_1_1_input_action}{Input\+Action} keys will give you entirely different input results.
\end{DoxyWarning}


\doxylink{class_atlas_1_1_input_action}{Input\+Action} contains a name (std\+::string), an key (int), a device type (\doxylink{namespace_atlas_ac597da9fe7e9baad81dcd94b291e5eae}{EInput\+Type}) and a trigger (\doxylink{namespace_atlas_a3de63d593b09a1492bb3d240f65ef88e}{EInput\+Trigger}) that will be used to distinguish between different Input\+Actions. Not only will it act as a key, but it will also act as a category.
