\doxysection{Atlas\+::File\+Manager Class Reference}
\hypertarget{class_atlas_1_1_file_manager}{}\label{class_atlas_1_1_file_manager}\index{Atlas::FileManager@{Atlas::FileManager}}


As the name suggests, this {\bfseries{non-\/copyable}} class is responsible for managing the files that \doxylink{namespace_atlas}{Atlas} and the user has loaded. This class has {\bfseries{3}} main responsibilities (excluding the janitor)\+: {\bfseries{Loading}}, {\bfseries{Unloading}} and {\bfseries{Registering}}. Additionally, this class can be configured to ignore certain patterns it comes across during the file discovery process through the use of {\bfseries{regex}} patterns.
