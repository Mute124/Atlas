\doxysection{Atlas\+::Vec3 Class Reference}
\hypertarget{class_atlas_1_1_vec3}{}\label{class_atlas_1_1_vec3}\index{Atlas::Vec3@{Atlas::Vec3}}


Represents a 3D vector. It will wrap raylib\textquotesingle{}s Vector3 and Jolt\textquotesingle{}s RVec3 to allow seamless integration. This also wraps raylib\textquotesingle{}s vector math functions. This class contains many functions because it wraps around Raymath\textquotesingle{}s functions and Jolt\textquotesingle{}s functions. Furthermore, there is alot of math that can be done with vectors, so this class tries to wrap as much as possible and make it easier to work with. The reason that this class exists despite raylib\textquotesingle{}s Vector3 and Jolt\textquotesingle{}s RVec3 is because there may be future plans to add different renderers.

\doxysubsection{Detailed Description}
Represents a 3D vector. It will wrap raylib\textquotesingle{}s Vector3 and Jolt\textquotesingle{}s RVec3 to allow seamless integration. This also wraps raylib\textquotesingle{}s vector math functions. This class contains many functions because it wraps around Raymath\textquotesingle{}s functions and Jolt\textquotesingle{}s functions. Furthermore, there is alot of math that can be done with vectors, so this class tries to wrap as much as possible and make it easier to work with. The reason that this class exists despite raylib\textquotesingle{}s Vector3 and Jolt\textquotesingle{}s RVec3 is because there may be future plans to add different renderers.
