\doxysubsection{Detailed Description}
Serves as a search key/input category for the input system. {\bfseries{Please pay attention when using this because any slight change in \doxylink{class_techstorm_1_1_input_action}{Input\+Action} keys will give you entirely different input results.}} 

\doxylink{class_techstorm_1_1_input_action}{Input\+Action} contains a name (std\+::string), an key (int), a device type (\doxylink{namespace_techstorm_a7ecb9dba2f9073f147a682772c49885d}{EInput\+Type}) and a trigger (\doxylink{namespace_techstorm_a3289ce49929ae6003fcbc78c954af69d}{EInput\+Trigger}) that will be used to distinguish between different Input\+Actions. Not only will it act as a key, but it will also act as a category.
