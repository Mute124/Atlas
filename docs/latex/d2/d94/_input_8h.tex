\doxysubsection{Detailed Description}
The input system for \doxylink{namespace_techstorm}{Techstorm}. 

The input system for \doxylink{namespace_techstorm}{Techstorm} works by a Laissez-\/faire system where the user can register Input\+Actions (Input\+Action) by using functions as callbacks. The reason for this is because it is much easier to do it this way (and it is also more efficient \& flexible). Plus, it also allows you to register multiple callbacks for the same Input\+Action, which may be useful.
