◆ EAudioDeviceType

enum class Atlas::EAudioDeviceType : int8_t
strong

This enum represents the different types of audio devices. It is also used to match the values of the ma_device_type enum, which is why PLAYBACK_DEVICE_TYPE is set to 1. This enum also inherits from int8_t because there are only 4 possible values.

Warning
If you are trying to set an audio device as a recording, please note that this is NOT the proper enum to use because this only applies to OUTPUT devices.

The values are as follows:

  • UNKNOWN_DEVICE_TYPE:
    With a value of -1, this represents an unknown device type. Typically, this is used when a device is not found, it is not initialized, or it is not available for some other reason.

  • PLAYBACK_DEVICE_TYPE: With a value of 1, this represents a playback device type.

  • DUPLEX_DEVICE_TYPE: With a value of 3, this represents a duplex device type.

  • LOOPBACK_DEVICE_TYPE: With a value of 4, this represents a loopback device type.
