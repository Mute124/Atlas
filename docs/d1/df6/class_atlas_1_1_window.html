◆ getNativeHandle()

std::any Atlas::Window::getNativeHandle ( )
overridevirtual

Returns a void* pointer of the native window handle. In the case of Windows, this will be a HWND.

Precondition
The window must have been initialized before this function can be called without issues.
Note
Should you be looking for the window handle for what ever window management API (ie. GLFW, SDL, etc), use the getNativePlatformHandle() function instead.
Returns
The native window handle as an std::any. This should be a glfwWindow* for GLFW and SDL_Window* for SDL.
Since
v0.0.9

◆ getNativePlatformHandle()

std::any Atlas::Window::getNativePlatformHandle ( )
overridevirtual

Returns the native window handle as an std::any for the underlying platform. For example, it will return a HWND for win32.

Remarks
As an example of how this function can be used, see the below example code (on a win32 platform):
try{
HWND nativeHandle = std::any_cast<HWND>(getNativeHandle());
} catch (const std::bad_any_cast& e) {
// handle error
}
