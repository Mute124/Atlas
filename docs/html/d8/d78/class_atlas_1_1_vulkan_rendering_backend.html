Detailed Description

A wrapper class for the usage of Vulkan as a rendering backend. This class handles the lifecycle of Vulkan and simplifies the usage of Vulkan to a simple API.

Precondition
The window must be set up and open prior to initializing Vulkan (and therefore this class!).
Warning
This class requires the user's device to support Vulkan!
Since
v0.0.1

Atlas::VulkanRenderingBackend::VulkanRenderingBackend ( )
default

Default constructor.

Note
This constructor is not intended to be used because the mAppInfo and mCreateInfo are required to create a Vulkan instance (and they are declared as a const!). Additionally, the variable mCreateInfo requires information on extensions and such, and since Atlas does not mandate any specific window management system (ie. GLFW), the mAppInfo MUST be set prior to calling this.
