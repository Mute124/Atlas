Detailed Description

This class represents a physical device that can provide haptic feedback to the user. Due to the sheer amount of ways haptic feedback can be provided, this class acts as a wrapper for the SDL_Haptic object.

Precondition
As is with the rest of Atlas' haptic API, SDL must be initialized before this class can be used and the flag SDL_INIT_HAPTIC must be set.
