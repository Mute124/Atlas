Detailed Description

template<typename T_KEY, typename T_SET_TYPE, typename T_CONTAINER, typename T_ENTITY_TYPE, typename T_HASH = std::hash<T_KEY>, typename T_EQUAL = std::equal_to<T_KEY>>
class Atlas::Registry< T_KEY, T_SET_TYPE, T_CONTAINER, T_ENTITY_TYPE, T_HASH, T_EQUAL >

This concept is used to ensure that the type is either a std::unordered_set or std::set.

Template Parameters
T_SET_TYPEThe type to check. This must be either a std::unordered_set or std::set.

This class serves as a base class for all registry classes. It is able to lookup entities by their ID, remove them, add them, iterate over them, and check if they exist.

Template Parameters
T_KEYThe type of the ID of the entity (what it should be stored as in the set and the container).
T_SET_TYPEThe type of the set used to store the entities. This should be either std::unordered_set or std::set.
T_EQUALThe type of the equality function used to compare the IDs of the entities.
T_CONTAINERThe type of the container used to store the entities. This should be either a std::map, std::unordered_map, std::multimap, std::unordered_multimap, std::vector.
T_HASHThe type of the hash function used to hash the IDs of the entities.
