Detailed Description

Common audio types.

This file contains common audio types like Atlas::InternalAudioDevice and Atlas::InternalAudioContext. The functionality depends on the audio backend. For example, if the audio backend is ATLAS_AUDIO_BACKEND_MINIAUDIO, then the InternalAudioDevice and InternalAudioContext are Miniaudio types.
