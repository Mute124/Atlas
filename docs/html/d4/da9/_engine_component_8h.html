class  Techstorm::abstract< K, D, T >
 This is an abstract interface singleton that all projects must implement and set the ProjectReference singleton to their project. If this boilerplate code is not done, Techstorm will not know what to run. In terms of functionality, this interface just holds the functions and variables that Techstorm will call (besides some initialization).
