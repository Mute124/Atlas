Detailed Description

As the name suggests, this non-copyable class is responsible for managing the files that Atlas and the user has loaded. This class has 3 main responsibilities (excluding the janitor): Loading, Unloading and Registering. Additionally, this class can be configured to ignore certain patterns it comes across during the file discovery process through the use of regex patterns.
