Detailed Description

Serves as a search key/input category for the input system.

Warning
Please pay attention when using this because any slight change in InputAction keys will give you entirely different input results.

InputAction contains a name (std::string), an key (int), a device type (EInputType) and a trigger (EInputTrigger) that will be used to distinguish between different InputActions. Not only will it act as a key, but it will also act as a category.
