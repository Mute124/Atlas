Detailed Description

template<typename T_VARIABLE_TYPE, typename T_MUTEX = std::shared_mutex>
class Atlas::ThreadSafeVariable< T_VARIABLE_TYPE, T_MUTEX >

This class wraps a variable (specified by T_VARIABLE_TYPE) and makes it thread-safe.

When ThreadSafeVariable::execute() is called, it will lock this variable and execute the function that is passed to it. Furthermore, it will also lock when the set() function, is called.

Warning
The variable that is wrapped by this class must have a default constructor. If the variable does not have a default constructor, then this class will throw an exception (MissingDefaultConstructorException).
Template Parameters
T_VARIABLE_TYPEThe type of variable to wrap. This is used as a variable in mValue.
T_MUTEXWhat type of mutex to use. This defaults to std::mutex
T_LOCK_GUARDWhat type of lock_guard to use. This defaults to std::lock_guard<T_MUTEX>
