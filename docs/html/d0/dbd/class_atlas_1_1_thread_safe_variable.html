Detailed Description

template<typename T_VARIABLE_TYPE, typename T_MUTEX = std::mutex, typename T_LOCK_GUARD = std::lock_guard<T_MUTEX>>
class Atlas::ThreadSafeVariable< T_VARIABLE_TYPE, T_MUTEX, T_LOCK_GUARD >

A managed variable is one that is thread-safe, handled by MemoryAllocator, and can be used to share data between threads.

Note
This is not used yet
Template Parameters
T_VARIABLE_TYPEWhat is the type of the variable

This class wraps a variable (specified by T_VARIABLE_TYPE) and makes it thread-safe. When ThreadSafeVariable::execute() is called, it will lock this variable and execute the function that is passed to it.

Template Parameters
T_VARIABLE_TYPEThe type of variable to wrap
T_MUTEXWhat type of mutex to use. This defaults to std::mutex
T_LOCK_GUARDWhat type of lock_guard to use. This defaults to std::lock_guard<T_MUTEX>
