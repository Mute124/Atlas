Detailed Description

An audio device is a physical OUTPUT device that can be used to play audio. If you are looking for a INPUT device, see AudioCaptureDevice. If you are looking for documentation on the forms of OUTPUT devices, see this class's details documentation.

Note
Please also note that this class's device type defaults to EAudioDeviceType::Playback.
Details

Audio devices come in a few forms:

  • Playback: An audio device that can be used to play audio.
  • Duplex: An audio device that can be used to both play and record audio.
  • Loopback: An audio device that can be used to record audio from another device.
