Detailed Description

The input registry is used to register InputActions (InputAction) and their callbacks by using a Laissez-faire system.

Note
Remember that this is a singleton, so you may use this statically.

Registers a pointer of InputAction to the callback given. This works as a key-value pair, where the key is the action and the value is the callback and is the reason why the return value is important.

Warning
Please do not lose it or you will not be able to unregister the callback or you may unregister the wrong one!
