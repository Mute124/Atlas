int MarchingCubes::calculate_cube_index ( GridCell &  cell,
float  isovalue 
)

Given a cell, calculate its cube index The cube index is an 8-bit encoding. Each bit represents a vertex. index[i] is the ith bit If the value at the ith vertex is < isovalue, index[i] = 1. Else, index[i] = 0
