Detailed Description

This is the interface for the user's project. This is the only interface that Atlas will call.

This is an abstract interface singleton that all projects must implement and set the ProjectReference singleton to their project. In terms of functionality, this interface just holds the functions and variables that Atlas will call (besides some initialization).

Note
If this boilerplate code is not done, Atlas will not know what to run.
Warning
Do not directly call window code in here or your project will crash!
